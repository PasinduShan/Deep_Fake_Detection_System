Overview

De-Sight is an AI-powered deepfake detection system designed to identify and prevent manipulated digital content. Using advanced machine learning algorithms, it analyzes images and videos to distinguish between real and synthetic media, ensuring authenticity and safeguarding against misinformation.

Github Repository

System Architecture

The deepfake detection system begins with data input, where users upload an image or video for analysis. The system preprocesses the data by resizing, normalizing, and extracting frames if necessary. Next, it performs feature extraction, detecting facial landmarks and inconsistencies in texture, lighting, or blinking patterns. The data is then analyzed by a deep learning model (e.g., CNN, EfficientNet, or Xception), which classifies the content as real or fake based on learned patterns. The system provides a confidence score along with visual indicators of manipulation. Finally, results are displayed to the user, and detected deepfakes can be logged for further analysis and reporting.
